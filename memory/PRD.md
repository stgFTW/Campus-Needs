# Campus Needs - Product Requirements Document

## Overview
Campus Needs is a marketing and browsing website for a hyper-local, student-to-student marketplace at the University of San Francisco. All CTAs redirect to the external marketplace at https://www.trickly.io/market/CampusNeeds.

## Brand
- **Primary**: USF Green (#006341)
- **Accent**: USF Gold (#C99700)
- **Supporting**: Dark Green (#004D32)
- **Background**: White
- **Font**: Inter

## Pages
1. **Homepage** — Hero with campus image, 4 category cards, How It Works (3 steps), CTA banner
2. **Categories** — 2×2 grid of category cards with green page header
3. **How It Works** — Detailed 3-step process + Providers/Buyers audience columns
4. **About** — Mission statement, 4 categories, Why P2P (Trust/Local/Affordable), Problem section

## Key Features
- All CTA buttons link to external marketplace
- Sticky navigation header with active state
- Mobile-first responsive design with hamburger menu
- Smooth scroll-to-section for How It Works anchor
- Scroll-to-top on page navigation
- Framer Motion entrance animations

## Tech Stack
- React + React Router
- Tailwind CSS with HSL design tokens
- shadcn/ui components (Button, Card)
- Framer Motion for animations
- Lucide React for icons
